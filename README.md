# Final Project – BI & BA for Decision-Making (MBA IBMEC)
**Data carnival – Power BI Dashboard**

This repository contains the final project developed for the course “Data Analysis and Decision-Making with BI and BA”, part of the MBA in AI, Data Science and Big Data for Business (IBMEC).
The deliverable is an interactive Power BI dashboard showcasing the complete BI workflow (ETL → Modeling → DAX → Visualization).

## Project Overview

The goal of this project is to transform raw operational data into clear, objective business insights.
The dashboard consolidates the main KPIs, comparisons, and trends required for the analytical scenario proposed in the assignment.

## Methods & Tools
### Power BI Desktop

ETL with Power Query

Data modeling (star schema orientation)

Creation of DAX measures and KPIs

Interactive visuals with cross-filtering and drilldowns

### Main Techniques

Data cleaning and transformation

Relationship modeling

KPI construction

Dashboard storytelling
